# Accepting User Input
In Java there are broadly two ways to take input from the user. The first way is to use BufferedReader Class and the second way is to use Scanner Class.

## Taking input using BufferedReader Class
BufferedReader is a class which id found in 'io' package. So, in order to use BufferedReader class we import 'io' package in our program

There is an function "readline()" which is used to read one line of input.

## Taking input using Scanner Class
Scanner is a class which is found in 'util' package. So, in order to use Scanner class we import 'util' package in our program.

There is a function called "next()" , "nextInt()" , "nextFloat()" etc whicih is used to read user input.

next() - reads in string format
nextInt() - reads in int format
nextFloat() - reads in float format
